# PortfolioHaris

Production-ready futuristic portfolio website for Haris Khan.

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Framer Motion
- GSAP
- React Three Fiber
- Lenis
- Lucide React

## Local Setup

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
npm run preview
```

## Netlify Deploy

Build output is generated in `dist`.

For drag-and-drop deployment, upload the `dist` folder to Netlify.

This project also includes:

- `netlify.toml`
- `public/_redirects`
- `public/sitemap.xml`
- `public/robots.txt`
- SEO meta tags and structured data in `index.html`
